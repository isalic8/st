###  Installation

~~~bash

sudo make install
~~~


### Some useful stuff

<C-UP> Scrolls up the terminal buffer

<C-DOWN> Scrolls down the terminal buffer

<C-LEFT> Zooms out

<C-RIGHT> Zooms in
