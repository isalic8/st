###  Installation

~~~bash
git clone https://gitlab.com/peter-117/st-simple
cd st-simple
sudo make install
~~~


### Some useful stuff

<C-UP> Scrolls up the terminal buffer

<C-DOWN> Scrolls down the terminal buffer

<C-LEFT> Zooms out

<C-RIGHT> Zooms in
